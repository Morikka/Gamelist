# Game List

Game list is a simple React project with a Notion database backend.

# Try IT

https://game.morikka.me

# Gatting Started

Set up Notion account and create a Notion database. Apply Notion API.

## Install dependencies

`yarn install`

## Run server

`yarn server`

## Run game list

`yarn start`

# Built with
+ React.js
+ Node.js
+ Notion API
